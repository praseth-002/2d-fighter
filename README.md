# Campus Arena Duel Tournament

**Campus Arena Duel Tournament** is a 2D fighting game developed using **Unity**.  
The game focuses on fast-paced local multiplayer combat, featuring hand-drawn characters and campus-inspired stages.

Designed as a school project, the game emphasizes core fighting mechanics, animation, and local PvP gameplay.
